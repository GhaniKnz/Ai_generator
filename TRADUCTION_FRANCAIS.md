## Terminologie Technique Traduite

### Statuts
- Running → En Cours
- Completed → Terminées
- Pending → En Attente
- Failed → Échouées

### Actions
- Start → Démarrer
- Pause → Pause
- Stop → Arrêter
- Refresh → Rafraîchir
- Create → Créer
- Manage → Gérer
- Monitor → Surveiller

### Concepts IA
- Training → Entraînement
- Dataset → Ensemble de Données
- Model → Modèle
- Workflow → Workflow (conservé)
- Generation → Génération
- Image → Image
- Video → Vidéo

### Interface
- Dashboard → Tableau de Bord
- Settings → Paramètres
- Assets → Ressources
- Monitoring → Surveillance
- Real-Time → Temps Réel
- Auto-Refresh → Rafraîchissement Auto
